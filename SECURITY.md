# Security Policy

This project is a documentation and Codex-skill package. It does not execute network services, collect credentials, or ship runtime code beyond validation tooling.

## Reporting A Concern

If you find a security or supply-chain concern, please open a GitHub issue with:

- the affected file,
- the risk,
- a minimal reproduction or explanation,
- the proposed fix if you have one.

Do not include secrets, tokens, private keys, or private repository contents in public issues.
