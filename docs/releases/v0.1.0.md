# v0.1.0

Initial public release of `andrej-karpathy-skills-for-codex`.

## Highlights

- Adds the `karpathy-guidelines-for-codex` Codex skill.
- Packages the skill with a Codex plugin manifest.
- Includes a Custom Instructions version for users who do not want to install a skill.
- Adds explicit credits and independence notices.
- Adds repository docs suitable for public collaboration.
